* Css

Css attributes are:

nameOfAttribute = "cssProperty:valueOfProperty"
** Css File Path

In html file, give path of css file, so it can be applied.

#+BEGIN_SRC html
 <link rel="stylesheet" href="./style.css" />
#+END_SRC

** Inline Css

- It is css in same line in specific html element.

Example: Make whole html documents background blue.

#+BEGIN_SRC html
<html style = "background:blue">
</html>
#+END_SRC

- It is not recommended to use inline styling for entire document.

** Internal Css

- This is done with special html tag calles style.

#+BEGIN_SRC html
<html>
  <head>
    <style>
      html{
      background:red;
      }
    </style>
  </head>
</html>
#+END_SRC

- This styling is limited to the html it sits in. If you have multipage html, you should be using external css not internal css like this one.

** External Css
- Css code is located inside an .css file.

Example: styles.css

#+BEGIN_SRC css
html{
    background:red;
}
#+END_SRC

- Link .css file to html file.

#+BEGIN_SRC html
<html>
  <head>
    <link
      rel="stylesheet"
      href="./styles.css"
    />
  </head>
</html>
#+END_SRC

** Css Selectors
*** Element Selector
- Select all h2 elements and make them red.

Example: styles.css

#+BEGIN_SRC css
h2{
    color:red
}
#+END_SRC

*** Class Selector

- . is class in css. red-heading is actual name of the class.
- class is an attribute that we can add to any html element. Example h2. It is like saying let's group all h2 elements into a class.

#+BEGIN_SRC css
.red-text{
    color:red;
}
#+END_SRC

In html code:

#+BEGIN_SRC html
<h2 class="red-text">Red</h2>
#+END_SRC

- this will select only element that hass class tag and turn it to red. It will not select and change color of other h2 tags that doesn't has class tags.
- css class tag and html class text must be same.
- Once defined in css class, it can be referred from anywhere. Let's say if we want to use red-text class inside <p> we can.
#+BEGIN_SRC html
<p class="red-text"> Paragraph </p>
#+END_SRC

*** Id Selector

- It works similirly to class selector. Difference between class and id selector is:

1. class selector can be applied to many elements
2. id can be applied to only one element in a single html.

#+BEGIN_SRC css
#main{
    color:red;
}
#+END_SRC

In html code:

#+BEGIN_SRC html
<h2 id="main">Red</h2>
#+END_SRC

*** Attribute Selector

- We can add many attributes to an html tag.

Example: Select all paragraph elements with attribute draggable and apply red color to it.

#+BEGIN_SRC css
p[draggable]{
    color:red;
}
#+END_SRC

In html:

#+BEGIN_SRC html
<p draggable="true"> Drag me</p>
<p draggable="false"> Don't Drag me</p>
#+END_SRC

If we want, we can add more rules. All paragraph tags that has draggable="false" to be red. It will not make other draggable ones red, because they don't have "false" value.

#+BEGIN_SRC css
p[draggable="false"]{
    color:red;
}
#+END_SRC

*** Universal Selector

- This will apply every element in the html. It will make everything red.

#+BEGIN_SRC css
*{
    color:red;
}
#+END_SRC

*** Box Properties

**** Border

We can change border size of a box.

#+BEGIN_SRC css
border : 10px solid black;
#+END_SRC

- 1opx thickness of the border
- solid is style of border. It can be dashed also
- black is color of the border.

We can change border of one side of the box in lower line.

#+BEGIN_SRC css
border : 10px solid black;
border-top: 0px;
#+END_SRC

This will keep 3 sides of box same but remove border from top.

We can also define border width depends on side:
border-width: 0px 10px 20px 30px

**** Padding

Padding will create gap between text and the border.

#+BEGIN_SRC css
padding: 5px
#+END_SRC

Let's say we have an image, if we add padding to its box, it will add space inside image and its box.

**** Margin

Margin is distance between 2 elements.

#+BEGIN_SRC css

}
#+END_SRC

#+BEGIN_SRC css
margin: 5px
#+END_SRC

**** How do we create a box? Div
- Div tag is how we create an artificial box around element(s).
- Div tags are invisable unless you add css to them. Once added, it is an invisable box that has content inbetween opening and closing tags.

#+BEGIN_SRC css
<div> Content </div>
#+END_SRC

#+BEGIN_SRC html
<body>

  <div>
    <p> Hello world </p>
    <img src="./world.png" />
  </div>

</body>
#+END_SRC
